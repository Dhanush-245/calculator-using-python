# Python Calculator

A small Python calculator project with two versions:

- `calculator.py` - a simple command-line calculator
- `scientific calculator.py` - a Tkinter scientific calculator with a graphical interface

## Requirements

- Python 3
- Tkinter, which is included with most Python installations

## How to Run

Run the simple command-line calculator:

```bash
python3 calculator.py
```

Run the scientific calculator:

```bash
python3 "scientific calculator.py"
```

## Scientific Calculator Features

- Addition, subtraction, multiplication, and division
- Brackets for grouped expressions
- Powers and square roots
- Logarithm and natural logarithm
- Trigonometric functions: sin, cos, tan
- Inverse trigonometric functions: asin, acos, atan
- Factorial
- Constants: pi and e
- Degree/radian mode toggle

## Notes

If the graphical calculator does not open, make sure your Python installation includes Tkinter.
